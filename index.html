Prototype Powertrain Team Member

University of Toronto Supermileage Team

Design printed circuit board (PCB) using Altium Designer to implement 22V motor controller for team's first fully electric car participating in Shell Eco-Marathon Americas Competition. Sourced electrical components (microcontrollers, amplifiers, encoders, drivers) to support cost-efficient construction.

Software Developer

UofT - Software Design and Communication

Developed a full stack C++ graphical mapping application with support for 20 international cities. Utilized OpenStreetMap API to provide local information on available transit options and points of interest. Implemented search algorithms including Dijkstra, A*, and Nearest Neighbour to search for optimal point-to-point travel routes and directions within 0.1 seconds. Integrated search autocompletion and intuitive transit layer with highlighting to improve client usability.

FPGA Designer

UofT - Digital Systems; Computer Organization

Developed a 16-bit processor using digital logic (finite state machines, registers, ALU, memory connections) to support 15+ ARM Assembly instructions. Utilized Intel Quartus Prime to integrate processor with DE1-SoC board, enabling functionality with HEX displays, switches, pushbutton keys, and other peripherals. Simulated programs with ModelSim to validate functionality with self-created testbenches.
